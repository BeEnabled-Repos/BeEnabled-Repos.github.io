Opening the door for all developers

Giving people with limited keyboard capabilities access to common coding languages.

  • Necessary skills: A point & tap user interface that invites the nearly 15% of Americans with a physical functioning difficulty into the rapidly growing tech job market.
  • Visual design: A coding language built with speech-to-text integration, controlled by simple commands, and has features like error highlighting & collapsing.
  • Complex ideas: The app can expand to allow for more advanced coding.
